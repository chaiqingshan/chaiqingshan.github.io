title:  JSON.parse()和JSON.stringify()
category: 技术
tags: [ 'js']
---
parse用于从一个字符串中解析出json对象,如

    var str = '{"name":"chaiqingshan","age":"23"}'

    JSON.parse(str)

结果：

    Object

    age: "23"
    name: "huangxiaojian"
    __proto__: Object



stringify()用于从一个对象解析出字符串，如


    var a = {a:1,b:2}

    JSON.stringify(a)

结果：

    "{"a":1,"b":2}"
